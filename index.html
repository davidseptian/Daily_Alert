Pengingat

Catat kegiatan, dapatkan reminder otomatis ke Telegram lewat cron-job.org. Lihat panduan di bawah.

Pengaturan Telegram & panduan singkat
Cara dapat token & chat id:
1. Di Telegram, chat ke @BotFather → kirim /newbot → ikuti instruksi → dapat token.
2. Cari bot kamu sendiri → tekan Start (kirim pesan apa saja).
3. Buka di browser: https://api.telegram.org/bot<TOKEN>/getUpdates → cari angka setelah "chat":{"id": — itu chat id kamu.

Cara pakai per-note: tiap note akan menghasilkan satu URL siap-pakai. Salin URL itu ke cron-job.org (gratis, daftar dengan email), buat job baru, atur jadwal sesuai tanggal & jam note, tempel URL, simpan. Saat waktunya tiba, cron-job.org akan memanggil URL itu dan Telegram bot otomatis kirim pesan ke kamu — tanpa perlu tab ini terbuka.
